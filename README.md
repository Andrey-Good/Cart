# Маятник на тележке

Численная модель системы `тележка + маятник` с интегрированием методом Рунге-Кутты 4-го порядка.

## Что есть в проекте

- `pendulum.py` — модель, интегратор и вспомогательные функции;
- `solve.py` — расчет двух сценариев и вывод численного результата в консоль;
- `simulate.py` — анимация движения в `pygame`.

## Сценарии

- `closed` — свободное движение системы без внешнего управления;
- `control` — стабилизация перевернутого маятника на тележке.

## Зависимости

- Python 3.12+
- `numpy`
- `pygame`
- `scipy`

## Запуск

```powershell
uv venv
uv sync
uv run python solve.py
uv run python simulate.py
```

Режим анимации задается константой `MODE` в `simulate.py`.
